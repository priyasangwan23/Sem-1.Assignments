Code Examples

Here are “Hello World” programs in HTML, C, and JavaScript:

HTML


<html>
  <body>
    <p>Hello World</p>
  </body>
</html>

C


#include <stdio.h>

int main() {
    printf("Hello World\n");
    return 0;
}

JavaScript


<script>
  console.log("Hello World");
</script>
